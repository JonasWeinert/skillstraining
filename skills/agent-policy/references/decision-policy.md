# Decision Policy Reference

Use this rubric in every Lumora exercise.

## ASK

Stop and ask when a choice changes the substance of the result:

- unit of analysis;
- eligibility, treatment, or outcome definition;
- duplicate resolution beyond the training default;
- sample restrictions;
- whether to describe a result as causal;
- any overwrite of raw data or a participant source deck.

## DEFAULT + Flag

Proceed with a stated default when the choice is conventional and reversible:

- household as the master unit;
- Stata-generated figures;
- descriptive language for welfare changes;
- output logs in `outputs/qa_reports/`.

## DOCUMENT

Record inferred choices:

- files inspected;
- merge cardinality;
- duplicate counts;
- filter counts;
- slide-to-source mappings.

## PROCEED

Always inspect inputs first, keep raw data immutable, validate IDs, and trace every number to a generated table or figure.
